# test_task_unicorn_
Калькулятор кошелька: Получает значение валют по отношению к доллару. Имеет возможность пополнения баланса(http://127.0.1.1:8080/modify), фиксировать значение баланса кошельков (http://127.0.1.1:8080/set), по запросу: 'http://127.0.1.1:8080/amount/get' конвертирует и выводит баланс в 3 валютах(рубли, доллары, евро).Можно получить значение курса к доллару поп запросу('valute'/get). Проверяет изменения баланса раз в минуту. На входе может принимать стартовые значения баланса для каждой валюты(--rub, --usd, --eur), доп параметры --debug(True - включен вывод в консоль, False - выключен вывод в консоль) и --period(кол-во минут через которое обновляется курс всех валют). Реализован вывоод информации в консоль при изменении валюты или баланса кошелька. Реализовано логирование.
